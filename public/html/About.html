技术

整个聊天互动游戏是基于html5编写。使用Koa处理后端的GitHub登录，Koa是下一代的 Node.js 的 Web 框架。由 Express 团队设计。旨在提供一个更小型、更富有表现力、更可靠的 Web 应用和 API 的开发基础。
服务器是通过Node.js实现的，node 是一个基于 V8 引擎的 Javascript 运行环境，它使得 Javascript 可以运行在服务端，直接与操作系统进行交互，与文件控制、 网络交互、进程控制等。简单的说node.js就是运行在服务端的 JavaScript。
